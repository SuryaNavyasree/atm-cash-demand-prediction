# ATM Cash Demand Forecasting

## Project Overview
This project predicts ATM cash demand using historical transaction data.

## Project Workflow

Data Collection → Data Cleaning → Data Analysis → Feature Engineering → Model Training → Prediction → Visualization

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Steps
1. Data Collection
2. Data Cleaning
3. Data Analysis
4. Forecasting Model
5. Prediction

## Goal
To help banks predict ATM cash requirements and avoid cash shortages.
## Results
The machine learning model predicts ATM cash demand based on historical transaction data, helping banks manage cash availability efficiently.
